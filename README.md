# Customer Churn Analysis & Retention Strategy

## 📌 Problem Statement
Customer churn directly impacts revenue and growth in subscription-based businesses.

## 🎯 Objectives
- Measure churn rate
- Identify churn drivers
- Estimate revenue loss
- Segment customers by risk

## 🛠 Tools Used
- SQL
- Python (Pandas, NumPy)
- Matplotlib, Seaborn

## 📊 Key Insights
- Low engagement leads to high churn
- Low satisfaction = high churn probability
- Frequent support tickets indicate churn risk
- Revenue loss concentrated in high-value customers

## 📈 Features
- End-to-end data cleaning
- KPI calculation using SQL
- EDA with visualization
- Churn risk scoring model
